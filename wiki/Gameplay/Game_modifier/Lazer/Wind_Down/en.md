# Wind Down (lazer mod)

![WD mod icon](/wiki/Gameplay/Game_modifier/Lazer/Mods/WD.png "Wind Down (WD) mod icon")

*For the full list of all lazer mods, see: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier/Lazer)*

## About

- Abbreviation: WD
- Type: Fun
- Score Multiplier: 0.50x
- Caption: `Sloooow doooown...`
- Compatible game modes: ![][osu!] ![][osu!taiko] ![][osu!catch] ![][osu!mania]
- Incompatible mods: [Half Time (HT)](/wiki/Gameplay/Game_modifier/Lazer/Half_Time), [Daycore (DC)](/wiki/Gameplay/Game_modifier/Lazer/Daycore), [Double Time (DT)](/wiki/Gameplay/Game_modifier/Lazer/Double_Time), [Nightcore (NC)](/wiki/Gameplay/Game_modifier/Lazer/Nightcore), [Wind Up (WU)](/wiki/Gameplay/Game_modifier/Lazer/Wind_Up), [Adaptive Speed (AS)](/wiki/Gameplay/Game_modifier/Lazer/Adaptive_Speed)
- Status: **UNRANKED**

## Description

The **Wind Down** mod progressively decreases the map speed starting with the default one (1.00x) and ending with the Half Time speed (0.75x).
