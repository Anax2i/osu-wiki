# Daycore (lazer mod)

*For the full list of all lazer mods, see: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier/Lazer)*

## About

- Abbreviation: DC
- Type: Difficulty Reduction
- Score Multiplier: 0.30x
- Default shortcut key: `R`
- Caption: `Whoaaaaa...`
- Compatible game modes: ![][osu!] ![][osu!taiko] ![][osu!catch] ![][osu!mania]
- Incompatible mods: [Half Time (HT)](/wiki/Gameplay/Game_modifier/Lazer/Half_Time), [Double Time (DT)](/wiki/Gameplay/Game_modifier/Lazer/Double_Time), [Nightcore (NC)](/wiki/Gameplay/Game_modifier/Lazer/Nightcore), [Adaptive Speed (AS)](/wiki/Gameplay/Game_modifier/Lazer/Adaptive_Speed), [Wind Up (WU)](/wiki/Gameplay/Game_modifier/Lazer/Wind_Up), [Wind Down (WD)](/wiki/Gameplay/Game_modifier/Lazer/Wind_Down)
- Status: **RANKED**

## Additional Information

The **Daycore** mod is exactly the same speed as [Half Time](/wiki/Gameplay/Game_modifier/Lazer/Half_Time), except that the audio has a lower pitch. Its opposite would be [Nightcore](/wiki/Gameplay/Game_modifier/Lazer/Nightcore) with [Double Time](/wiki/Gameplay/Game_modifier/Lazer/Double_Time).
