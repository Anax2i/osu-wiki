# No Release (lazer mod)

*For the full list of all lazer mods, see: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier/Lazer)*

## About

- Abbreviation: NR
- Type: Difficulty Reduction
- Score Multiplier: 0.90x
- Default shortcut key: `T`
- Caption: `No more timing the end of hold notes.`
- Compatible game modes: ![][osu!mania]
- Incompatible mods: [Hold Off (HO)](/wiki/Gameplay/Game_modifier/Lazer/Hold_Off)
- Status: **UNRANKED**
