# Relax (lazer mod)

![RX mod icon](/wiki/Gameplay/Game_modifier/Lazer/Mods/RX.png "Relax (RX) mod icon")

*For the full list of all lazer mods, see: [Game modifier (lazer)](/wiki/Gameplay/Game_modifier/Lazer)*

*For the stable wiki, see: [Relax (stable)](/wiki/Gameplay/Game_modifier/Relax)*


## About

- Abbreviation: RX
- Type: Automation
- Score Multiplier: 0.10x
- Default shortcut key: `C`
- Caption:
  - ![][osu!]: `You don't need to click. Give your clicking/tapping fingers a break from the heat of things.`
  - ![][osu!taiko]: `No need to remember which key is correct anymore!`
  - ![][osu!catch]: `Use the mouse to control the catcher.`
- Compatible game modes: ![][osu!] ![][osu!taiko] ![][osu!catch]
- Incompatible mods: [Autoplay (AT)](/wiki/Gameplay/Game_modifier/Lazer/Autoplay), [Cinema (CN)](/wiki/Gameplay/Game_modifier/Lazer/Cinema), [Autopilot (AP)](/wiki/Gameplay/Game_modifier/Lazer/Autopilot), [Alternate (AL)](/wiki/Gameplay/Game_modifier/Lazer/Alternate), [Single Tap (SG)](/wiki/Gameplay/Game_modifier/Lazer/Single_Tap), [Magnetised (MG)](/wiki/Gameplay/Game_modifier/Lazer/Magnetised)
- Status: **UNRANKED**
